Quiz Deret

1. Apa nilai dari deret aritmatika berikut: 2, 5, 8, 11, ... ?

2. Berapakah suku ke-5 dari deret geometri berikut: 3, 6, 12, 24, ... ?

3. Berapakah jumlah 10 suku pertama dari deret aritmatika berikut: 1, 4, 7, 10, ... ?

4. Berapakah suku ke-7 dari deret geometri berikut: 2, 4, 8, 16, ... ?

5. Tentukan hasil penjumlahan dari deret tak hingga 4+12+36+108, ... ?

Penjelasan Penyelesaian:

Soal 1: Deret aritmatika 2, 5, 8, 11, ...

Dik : a = 2, b = 5 - 2 = 3

Dit : U5

Un = a + (n -1 )b

U5 = 2 + (5 -1) x 3

U5 = 2 + 12

U5 =14

Jadi, suku ke-5 dari deret aritmatika diatas adalah 14.

Soal 2: Deret geometri 3, 6, 12, 24, ...

Dik : a = 3, r = 6/3 = 2

Dit : S5

Sn = a (r^n-1)

S5 = 3 (2^5-1)

S5 = 3 (2^4)

S5 = 3 (16)

S5 = 48

Jadi, suku ke-5 dari deret geometri diatas adalah 48.

Soal 3: Deret aritmatika 1, 4, 7, 10, ...

Dik : a = 1, b = 4 - 1 = 3

Dit : Jumlah 10 suku pertama

Sn = n/2 (a + an)

an = a + (n - 1) b

a10 = 1 + (10 - 1) 3 = 1 + 27 = 28

S10 = 10/2 (1 + 28)

S10 = 5 (29)

S10 = 145

Jadi jumlah 10 suku pertama dari deret aritmatika diatas adalah 145.

Soal 4: Deret geometri 2, 4, 8, 16, ...

Dik : a = 2, r = 4/2 = 2

Dit : S7

Sn = a (r^n-1)

S7 = 2 (2^7-1)

S7 = 2 (2^6)

S7 = 2 (64)

S7 = 128

Jadi, suku ke-7 dari deret geometri diatas adalah 128.

Soal 5: Deret tak hingga 4+12+36+108, ...

Dik : U2 = 12, U1= 4

Dit : hasil penjumlahan

U2/U1 = 12/4 = 3
